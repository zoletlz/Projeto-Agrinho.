# SimãoAgro.✩°𓏲⋆🌿. ⋆⸜ ✮˚
Projeto para o concurso Agrinho. Site projetado para auxiliar agricultores de Campina do Simão PR.
# Objetivo༄˖°.🍂.ೃ࿔*:･:
O objetivo central do projeto SimãoAgro | Clima & Regeneração é democratizar o acesso à agricultura de precisão, transformando dados climáticos e ambientais complexos em decisões simples, práticas e ecológicas para o produtor rural de Campina do Simão - PR.
# Tecnologias Usadas𓇼:
٠ ࣪⭑ Tecnologias Base (Nativas):
 
1. HTML5 (HyperText Markup Language).⋆˚✿˖°
2.  CSS3 (Cascading Style Sheets).⋆°✿˖°
3.   JavaScript (ES6+).⋆°✿˖°
   
   ٠ ࣪⭑ Bibliotecas de Terceiros (Open-Source):
   
1.   Leaflet.js: Uma das bibliotecas JavaScript mais leves e populares do mundo para mapas interativos.⋆°✿˖°
2.   Chart.js: Uma biblioteca baseada na tecnologia de renderização HTML5 <canvas>. Ela foi utilizada para gerar o gráfico de barras do Balanço Hídrico, convertendo arrays de dados numéricos em elementos visuais dinâmicos, interativos e fáceis de ler.⋆°✿˖°
   
   
  
   ٠ ࣪⭑ Recursos de Design e Tipografia:
   
1.   Google Fonts.⋆°✿˖°
2.   Fontes de Dados (Simuladas no código): Fatores de conversão reais baseados em diretrizes internacionais de balanço de Gases de Efeito Estufa (como os cálculos de emissão por litro de diesel e fixação de CO2 pelo manejo de palhada do Plantio Direto).⋆°✿˖°
